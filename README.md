## Installation Guide

- clone the repository
```bash
https://github.com/KawsarAhmad43/ex-3-coming-soon.git
```
- install the composer
```bash
composer install
```


- Configure the env - make a database and set this:
```bash
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ex3_temp
DB_USERNAME=root
DB_PASSWORD=
```

- Run the server
```bash
php artisan serve
```
